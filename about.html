Project Description

This Game Launcher is a class project designed to showcase our JavaScript skills. It features three different games:

  • Tic-Tac-Toe: A classic game where players take turns marking X and O on a 3x3 grid.
  • Rock Scissors Paper: A game of chance where you compete against the computer.
  • Snake: A nostalgic game where you control a snake to eat apples and grow without hitting yourself.

All games include local storage functionality to keep track of scores between sessions.
